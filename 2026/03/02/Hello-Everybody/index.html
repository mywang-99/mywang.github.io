little think

I want to start a blog. I rarely show interest in anything, but once I do, I want to pursue it until I’m satisfied, even if it might be boring or pointless. However, I know myself very well. If I don’t nurture this initial spark of interest during that time, it will soon fade, and I won’t be able to approach any task with the same level of enthusiasm. Let me balance my research work with my interests in this busy life. There must be some value in things simply because they exist.
