WSL2安装 | 搭建Ubuntu && Geant4 && ROOT

前期准备

我们都知道许多软件开发平台、数据处理依赖平台都是基于 Linux 操作系统，而现在大家接触到的电脑系统普遍都是 Windows 系统。为了解决现有 Windows 平台上急需使用 Linux 系统来平衡工作这一问题，以下是我对安装操作系统的一些拙见和安装过程中值得分享的关键。

WSL 提供了一个基于 Windows 系统的 Linux 操作平台，方便我们在使用 Linux 系统时，同时使用 Windows 跑一些程序。当然有人要疑惑了，下载一个虚拟机用于 Linux 系统或者在电脑上重装双系统不是更方便吗？

如果真正使用这两种方式进行数据处理后，就会发现安装镜像虚拟机会大大吃掉电脑性能并且抢用显卡和磁盘，而虚拟机致命弱点之一就是虚拟磁盘不可随意编辑和移动。这边也是一个本人的案例，我当时因为挪动了桌面文件导致虚拟磁盘崩溃打不开，即使创建新的虚拟机共用一个磁盘依旧打不开原始程序，另外就是用过的人没有人说过它的好。

而另一个在电脑上安装双系统这个方法，当然很好，Linux 系统本就是不可打败的强者，甚至于买一台 Linux 操作系统的电脑都是最明智的选择，但是研究生们已经被 Windows 同化了 4 年+，我想除了有导师愿意花足够的资金给理工科研究生安排主机之外，应该没人会主动选择安装双系统了吧。

注意事项：

  • 只有 Win10+（2004+）的系统才能使用 WSL
  • 保持网络畅通，最好连接网线，保持连接
